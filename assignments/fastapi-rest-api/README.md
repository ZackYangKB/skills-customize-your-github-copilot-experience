# 📘 Assignment: FastAPI REST API 开发

## 🎯 Objective

通过本次作业，你将学习如何使用 FastAPI 框架构建 RESTful API，掌握接口设计、数据处理和基本后端开发技能。

## 📝 Tasks

### 🛠️ 基础 API 构建

#### Description
使用 FastAPI 创建一个简单的 RESTful API，实现数据的增删查改（CRUD）功能。

#### Requirements
Completed program should:

- 搭建 FastAPI 项目结构
- 实现至少一个数据模型（如学生、任务等）
- 提供 GET、POST、PUT、DELETE 等接口
- 能够在本地运行并测试 API
- 返回规范的 JSON 响应

### 🛠️ 进阶挑战（可选）

#### Description
为你的 API 增加更多功能或优化体验。

#### Requirements
Completed program should:

- 支持数据验证和错误处理
- 添加简单的用户认证（如 token 验证）
- 编写 API 文档（可用 FastAPI 自动生成的 docs）
- 支持分页或筛选查询
